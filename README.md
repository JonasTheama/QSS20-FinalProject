Exploring the ICIJ Offshore Leaks database to trace how wealth moves 
around the world.

https://offshoreleaks.icij.org

## Code

  generates starter visualizations

## Output
- `output/viz1_officer_countries.png` — top countries of origin for officers
- `output/viz2_entity_jurisdictions.png` — top shell company jurisdictions
